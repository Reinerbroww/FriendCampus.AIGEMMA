# FriendCampus.AI

AI-powered study companion for university students, built with Gemma 4.

## Features
- Ask Anything — chat with Gemma about any subject
- Smart Roadmap — AI-generated hierarchical learning path
- Check Understanding — topic-specific comprehension testing
- Weakness Report — track which topics need more attention
- Academic References — web-powered reference search
- Image Analysis — upload photos of problems for AI explanation

## Tech Stack
- Backend: Python Flask
- AI: Google Gemma 4 (26B)
- Database: PostgreSQL (Supabase)
- Frontend: HTML, CSS, JavaScript
- Deployment: Railway

## Live Demo
https://friendcampus-ai-production.up.railway.app

## Setup
1. Clone repo
2. Install dependencies: pip install -r requirements.txt
3. Set environment variables (GEMINI_API_KEY, DATABASE_URL, SECRET_KEY)
4. Run: python app.py
